<h2>Описание ДЗ</h2>

<ul>Необходимо разработать классы для работы с вещественными числами, векторами (строками размерности 1xN и столбцами размерности Nx1) и матрицами размерности MxN и реализовать операции над ними:<ul>

<p>Что из этого можно понять ?</p>


1) Сколько классов ? - матрица, ветор строка и вектор столбец
2) Что будут содержить эти классы ? - std::vector<double> и набор методов для взаимодествия вещественными числами
3) Какие абстрации возможны ? - объект матричного калькулятора, который может быть чем угодно; абстракции вектора, который может чем угодно; может быть матрица 

а) Виртуальный класс, в котором переопределен метод создания матрицы (заполнения вектора).
б) Виртуальный класс матрица от которого наследуется матрица строка, матрица столбец и обычная матрица
